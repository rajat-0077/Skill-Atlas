Contribute to Skill Atlas

Skill Atlas relies on community experts to keep our roadmaps accurate and up-to-date. Here is how you can help.

Suggest a New Roadmap

Have a detailed plan for a skill or exam we don't cover? Submit your proposal below and our curation team will review it.

Suggest Better Resources

Found a broken link or know a better YouTube tutorial for an existing roadmap node? You can submit resource updates directly through our GitHub repository.

  1. Find the roadmap JSON file in the /data/roadmaps directory.
  2. Update the resource links array for the specific node.
  3. Submit a Pull Request with a brief explanation.
